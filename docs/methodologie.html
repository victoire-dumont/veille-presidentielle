Comment est construit le barometre economie circulaire

Principe

Le barometre compte le nombre de mesures programmatiques de chaque candidat en lien direct avec l'economie circulaire. Seuls les programmes officiels publies sont analyses. Les candidats sans programme publie apparaissent avec la mention "programme non publie".

Mots-cles recherches

Une mesure est comptabilisee si elle traite directement de l'un de ces sujets :

recyclage reemploi reutilisation eco-conception obsolescence programmee zero dechet consigne reparation durabilite des produits compostage economie circulaire
